## Common Commands

| Task | Command |
|------|---------|
| Install dependencies | `pip install -r requirements.txt` |
| Build AppImage | `python -m PyInstaller --onefile --add-data "assets;assets" --add-data "scrshots;scrshots" app.py && appimagetool ./dist/Protodesk-1.3.0-x86_64.AppImage` |
| Run locally (debug) | `python app.py` |
| Run a single test (if added) | `pytest <path/to/test_file.py>` |

## Linting

The project uses `flake8` for linting. Install and run:
```
pip install flake8
flake8 .
```

## Architecture Overview

* The app is a **Qt‑based** desktop client built with **PySide6**.
* `app.py` contains the main window (`ProtonDesktopApp`) and several dialog classes (`AboutDialog`, `DonateDialog`).
* A single `QWebEngineView` (`ProtonWebView`) embeds Proton services. The view loads Proton Mail by default and routes other Proton domains through the same view; external links are opened in the default browser.
* Downloads are handled by Qt's `QWebEngineDownloadRequest` and trigger a notification on completion via `show_notification`.
* UI styling is provided by `.qss` files (`about_dialog.qss`, `donate_dialog.qss`) and assets located in `assets/`.

## Development Tips

* The application state (login session, cookies) is persisted by Qt's default `QWebEngineProfile` under `~/.config/Protodesk`.
* For rapid iteration, run `python app.py` and edit source; the app must be restarted to pick up changes.
* Packaging with PyInstaller produces a single executable in `dist/`. The resulting AppImage can be found in the same directory.
